## What to include in the output

### 1. Survey intro text (Hebrew)
The text that appears at the top of the Google Form, before Q1. It should:
- Identify the researchers (students at Afeka College studying UI design)
- Explain the purpose without revealing too much that would bias answers
- Confirm anonymity
- State it takes ~3 minutes
- Mention the study is about mental health app experiences

### 2. All 8 questions with complete answer options
For each question, write:
- Question text (final Hebrew, cleaned up)
- Answer options (all of them, clearly listed)
- Question type: Single choice (בחירה יחידה) / Multiple choice (בחירה מרובה) / Scale (סקלה)
- Whether it is required (חובה) or optional (רשות)
- Any display logic notes (e.g., "show Q4 only if Q3 answer is 'Yes, regularly' or 'Sometimes'")

### 3. Closing text
What appears after the last question. Thank you message, mention that results will be presented anonymously.

### 4. Distribution copy
A short WhatsApp/Facebook message in Hebrew that Tamar can paste when sharing the survey link. Should be warm, short (~50 words), explain who it's for, take only 3 minutes.

## Quality requirements
- All Hebrew must be natural, not machine-translated
- Answer options must be mutually exclusive where required
- Answer options must cover all likely real-world responses (no "other" trap for main options)
- Q4 must have the "Multiple choice" type flag because respondents can select multiple
- Questions must not lead the respondent toward a particular answer
- The survey must feel respectful of the sensitive topic — not clinical, not scary
